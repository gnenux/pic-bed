# 图床服务程序

    基于beego & bootstrap4

### Usage:

1. 

    cd ./pic-bed/models
    vim ./models.go
    Change the line 32&34 to your own database account and password

2. 

    cd ./pic-bed
    go build
    ./pic-bed

or

    cd ./pic-bed
    go run ./main.go
